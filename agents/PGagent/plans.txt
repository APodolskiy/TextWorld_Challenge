Заметки по PGagent для TextWorld

### Необходимости:
1. Добавить поддержку батча в среде

### С admissible commands:
1. Добавить несколько состояний в наблюдение, например 4 последних
2. Изменить способ получения embedding-га состояния.
   Сейчас это среднее embedding-гов всех токенов в строке состояния, а нужно сделать взвешенную сумму.
   В качестве коэффициентов можно использовать tf-idf.
   Либо делать выбор тех entities, которые есть в глобальных entities и повышать их веса.
3. Придумать способ подключить рецепт. Один из возможных способов - attention рецепта и векторов токенов наблюдения.
4. Добавить RNN в модель policy, обдумать идеи, связанные с памятью

###  Без admissible commands
1. Можно использовать verbs и entities для построения команд. Здесь может быть применимо attention.
2. Другой способ - давать распределение по словарю на фразу из 2-5 слов.
3. Также можно попробовать генерировать команду с помощью LSTM.
Или вообще попробовать GAN. Но не очень понятно, как его обучать.

###  Общие замечания:
1. Трудности с обучением скорее всего будут с использованием любой архитектуры.
2. Наверное одно из самых главных - параллельно обучать агента.
Решить это можно, создав несколько агентов, которые обмениваются весами после нескольких эпох
